# AI-DS
Handwritten Digit Classifier (MNIST):

Developed an image processing solution to accurately identify and extract numerical data from blurred images. Utilized advanced techniques in computer vision, including image enhancement, deblurring algorithms, and Optical Character Recognition (OCR) to retrieve critical information from low-quality or distorted visuals. This project improved data extraction accuracy and demonstrated practical applications in document analysis and automated data processing.

Data Analytics:

This data analysis project focuses on understanding consumer buying behavior during the Diwali festival season. The primary objective is to derive insights from sales data to help businesses enhance their marketing strategies, improve customer targeting, and increase overall sales during festive periods.
